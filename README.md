## Install

`npm install px2rem-loader`

## webpack config

```
module.exports = {
  // ...
  module: {
    rules: [{
      test: /\.css$/,
      use: [{
        loader: 'style-loader'
      }, {
        loader: 'css-loader'
      }, {
        loader: 'px2rem-loader',
        // options here
        options: {
          remUnit: 75,
          remPrecision: 8,
           exclude:/antd\.css/
        }
      }]
    }]
  }
}
```


Please see [px2rem](https://github.com/songsiqi/px2rem) for more information about query parameters of px2rem.
